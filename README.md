# Recipely
_recipely is a recipe web application that helps users save there recipes in one place so that they can find them whenever they want.It's built using 
MEN Stack template based on authentication. 
![screenshot of the website](images/memory-cards.png)


### Features

1. **User authentication**: Sign-in, Sign-up and Sign-out securely. 
2. **Recipe management**: Create, Read, Update, Delete.
3. **Comments**: users can add comments on the recipes page. 
4. **Dashboard**:every user has different dashboard, that will shows the recipes they've created. 
5. **404 page**: when the users enter unmatched route.
6. **protected routes**: recipe creating, editing, deleting needs to be authenticated user. 

### Installation

No installation required! Simply clone the repository to your local machine and open the `server.js` file in your favorite browser to see the application.

```bash
git clone https://github.com/bynbdulla/recipely
cd recipely
open server.js
```
